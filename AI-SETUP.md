# Lal Ji Clinic AI Setup

1. Install dependencies:
   `npm install`
2. Create `.env` from `.env.example`
3. Add your OpenAI API key in `.env`
4. Start the backend:
   `npm start`
5. Keep the website open and refresh `index.html`

The website chat widget calls:
`http://localhost:3000/api/clinic-assistant`

The API key stays on the backend, not in the browser.
